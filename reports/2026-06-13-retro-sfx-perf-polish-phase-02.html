Sprint 5 Phase 02 Report

Isometric Renderer Performance Hardening

Tuned the heavier isometric canvas renderer so it stays smooth on mid-range and mobile devices with bounded resource use.

Date: 2026-06-13 Commit: 085c432 Tests: 92 passed

What Changed

Device Pixel Ratio Cap (web/js/isocity.js)

  • Added MAX_DPR = 2 so high-DPR mobile displays don't over-render the canvas.
  • Resize now sets integer canvas dimensions and resets the transform matrix before scaling to avoid drift.

Adaptive Particle Budget

  • Particle cap scales with plot count: min(150, 40 + plots.length * 8).
  • spawnParticles and burstParticles both guard against exceeding the cap.
  • Reduced smog spawn rate slightly (emissions / 800) to keep per-frame work predictable.

Debounced Resize

  • Window resize events are debounced to 100ms to prevent layout thrashing.
  • Timer is cleared on destroy().

Background Tab Throttling

  • RAF loop already pauses on background tabs; added an explicit document.hidden guard for state hygiene.
  • 30fps interval gate is unchanged and enforces the frame budget.
