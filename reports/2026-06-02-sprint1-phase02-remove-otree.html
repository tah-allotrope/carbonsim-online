06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Limitation: The archived platform at archive/otree-platform/runnable/ is no longer actively maintained. If Sprint 4 needs to reference it, the code may be stale relative to engine changes.
  • Limitation: The requirements.txt uses minimum version constraints. A future pip install may pull newer versions that introduce breaking changes. A lock file (e.g., requirements.lock) could be added later.
  • Second-best alternative: Delete platform/ entirely (per the original TASK-02-01). Rejected per Q-001 answer — user chose archive to keep the runnable platform available for Sprint 4 reference.
